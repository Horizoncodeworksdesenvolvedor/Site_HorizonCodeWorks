E-COMMERCE

Loja Virtual TechStore

HTML CSS JavaScript Node.js MySQL

O Mercado Digital é um sistema de e-commerce que oferece uma interface responsiva e intuitiva para gerenciar clientes (PF/PJ), produtos, pedidos e entregas. Inclui dashboard com estatísticas em tempo real, carrinho de compras, múltiplas formas de pagamento (PIX, cartão, boleto) e controle de estoque. A API RESTful gerencia operações com segurança, usando validações, transações MySQL e foreign keys. Configuração simples via npm install e script SQL. Ideal para pequenos negócios e aprendizado full-stack.

Sistema

Encrypted Guardian

Java

Uma ferramenta CLI que criptografa e descriptografa arquivos de texto (.txt), utilizando algoritmos como AES, César e Transposição. Além disso, ele mede o desempenho de cada método. Fundamentalmente, o sistema demonstra a implementação da criptografia para proteger dados em sistemas, assegurando a confidencialidade e integridade de informações sensíveis que transitam por esses arquivos.

Java MySQL

O Sistema de Gerenciamento de Alunos é uma ferramenta que simplifica o controle de informações e notas acadêmicas. Com uma interface intuitiva, organiza dados em tabelas por sala, atendendo às necessidades de professores e administradores.

Sobre a HorizonCodeWorks

A HorizonCodeworks é uma empresa fundada por três jovens estudantes de Ciência da Computação e Desenvolvimento de Sistemas, com foco em criar soluções tecnológicas sob medida. Especializada no desenvolvimento de websites, sistemas personalizados e plataformas digitais, a empresa atende lojas e empresas que buscam otimizar processos e modernizar suas operações. Com uma visão jovem e inovadora, a HorizonCodeworks se destaca por entregar projetos funcionais, intuitivos e alinhados às necessidades do cliente, contribuindo para sua transformação digital e crescimento.

Nossa missão é transformar ideias em soluções digitais que impulsionem o crescimento dos nossos clientes, através de tecnologias modernas e processos ágeis.
